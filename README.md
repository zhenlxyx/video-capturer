# video-capturer
 从视频中捕捉包含运动的画面

# 系统支持

Microsoft Windows 10 版本 1809 或更新

Python 3 以上

OpenCV 3 或以上

## 开始使用之前

1. 安装 Python 环境

   打开 Microsoft Store 应用，搜索“Python”并下载 Python 3.7 / 3.8。

2. 安装 OpenCV

   打开命令提示符，输入 pip install opencv-python 并回车，等待安装完成。

3. 安装 imutils 图像操作简便函数集

   打开命令提示符，输入 pip install imutils 并回车，等待安装完成。
